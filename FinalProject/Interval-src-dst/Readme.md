This MapReduce will create intermediate data.
The format is:
(interval), (src->dst) \t (times)

mapper4 and reduce4 :
You can use them to get
src->dst \t times
input of mapper should be the result of first mapreduce

mapper5 and reduce4 :
You can use them to get
season,dst \t times
input of mapper should be the result of first mapreduce
0 is winter 3 is fall
